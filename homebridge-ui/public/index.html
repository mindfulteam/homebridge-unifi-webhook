Incoming webhook URLs

Paste a sensor's URL into its UniFi Alarm Manager Webhook action.

Use an address your UniFi console can reach — not saved.
This config also requires the shared-secret header (Authorization: Bearer … or X-Webhook-Token: …) on every UniFi webhook action.
